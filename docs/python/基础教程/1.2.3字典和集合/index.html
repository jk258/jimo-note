字典和集合

字典​

字典是以键值对的形式进行存储数据的,以{key:value,...}表示
字典的key必须是可哈希的数据类型，字典的value可以是任何数据类型

dic = {
    "name": "test",
    "age": 12
}
print(dic["name"])#test

字典的操作​

添加修改​

通过key添加修改

dic = {
    "name": "test",
    "age": 12
}
dic["sex"]="男"
dic["name"]="test2"

使用setdefault,(设置默认值，如果以前已经有值了，setdefault就不起作用了)

dic = {
    "name": "test",
    "age": 12
}
dic.setdefault('sex','男')

删除​

pop根据key删除

dic = {
    "name": "test",
    "age": 12
}
dic.pop('age')

查找​

通过key或get(key)获取

dic = {
    "name": "test",
    "age": 12
}
print(dic['sex'])#如果key不存在，程序会报错
print(dic.get('sex'))#如果key不存在，程序返回None

循环​

  • 通过for循环，直接拿到key
dic = {
    "name": "test",
    "age": 12
}
for key in dic:
    print(key)
  • dic.keys()获取key值列表
  • print(dic.values())获取value列表
  • dic.items()获取[(key,value)]列表
  • 也可以for循环dic.items()
dic = {
    "name": "test",
    "age": 12
}
for key,value in dic.items():
    print(key,value)

集合​

set集合是无序的

s = {1, 2, '哈哈',3}
print(s)#{1, 2, 3, '哈哈'}

*set集合内应该是不可变类型

s = {1, 2, '哈哈', []}
print(s)  # TypeError: unhashable type: 'list'
  • 不可哈希：python中的set集合进行数据存储的时候，需要对数据进行哈希计算，根据计算出来的哈希值进行存储数据
    • set集合要求存储的数据必须是可以进行哈希计算的
    • 可变的数据类型,list,dict,set
  • 可哈希：不可变的数据类型，int,str,tuple,booll

创建空集合

s=set()

set集合操作​

  • 新增add
s=set()
s.add('赵')
s.add('王')
s.add('里')
print(s)#{'王', '里', '赵'}
  • 删除remove
s={'赵','王','里'}
s.remove('里')
print(s)#{'赵', '王'}
  • 修改：先删除，后添加
s={'赵','王','里'}
s.remove('里')
s.add('孙')
print(s)#{'赵', '孙', '王'}
  • 查询：for循环查询
s={'赵','王','里'}
for item in s:
    print(item)

set去重​

lst=['赵','王','里','赵','王','里','赵','王','里']
print(list(set(lst)))#['王', '赵', '里']

ps:去重后的列表是无序的

并集、差集和交际​

s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6, 7}
print('交集:', s1 & s2)#交集: {3, 4}
print('并集:', s1 | s2)#并集: {1, 2, 3, 4, 5, 6, 7}
print('差集:', s1 - s2)#差集: {1, 2}
